# Private Beta Quick Guide

Welcome to Hermes for Web.

Think of this product as a desk for your AI work:
- chat in the middle
- tools on the left
- files on the right

## What to try first
1. Start a new chat
2. Open the sidebar tool panels
3. Try one Setup Pack
4. Create one Artifact
5. Open the Workspace panel and inspect the file
6. Try a model switch

## Best first workflows
### If you use Obsidian
- run Obsidian Starter Pack
- create a note artifact
- try ShareNote + Telegram Pack

### If you are new to Telegram Hermes
- run Telegram Onboarding Pack
- ask Hermes to explain the first-message workflow
- try a Telegram handoff action from an artifact

### If you want to test quality/reliability
- run Note / Posting / Cron Preflight checks
- verify whether the guidance matches what you expect

## What feedback is most useful
Please tell us:
- where you got confused
- where the UI felt too crowded
- where you expected one click but needed three
- whether Setup Packs felt helpful or vague
- whether the assistant name / personalization made the UI feel more yours

## The key question
Does this feel like a chat app,
or does it feel like an actual AI workspace?

That answer matters more than any single bug.
