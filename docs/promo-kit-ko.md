# Hermes for Web 홍보 세트

## 1. Private beta 초대문
안녕하세요!
Hermes for Web private beta 를 공유드립니다.

repo:
https://github.com/reallygood83/hermes-for-web

이 프로젝트는 단순한 AI 채팅창이 아니라,
- 사용자의 Hermes memory 를 읽고
- Setup Packs 로 환경을 시작하고
- Artifact 로 결과물을 남기고
- Telegram / Obsidian / CLI 흐름까지 이어주는
개인화 AI 작업실을 목표로 합니다.

특히 이런 분들께 잘 맞습니다.
- Obsidian 중심으로 일하는 분
- CLI / Telegram / WebUI 를 함께 쓰는 분
- 대화를 결과물 파일로 남기고 싶은 분
- 로컬 모델과 원격 모델을 함께 쓰는 분

추천 첫 사용 순서:
1. 봇 이름 설정
2. Setup Pack 하나 실행
3. Artifact 하나 생성
4. Workspace / Telegram handoff / Share 흐름 확인

피드백은 “무엇이 좋았는지”보다
“어디서 멈칫했는지”가 더 중요합니다.

## 2. GitHub 소개문
Hermes for Web 은 기억하고, 만들고, 이어주는 개인화 AI 작업실입니다.

이 프로젝트는 Hermes Agent 를 위한 workflow-oriented WebUI 포크로,
단순 채팅이 아니라 setup packs, artifacts, workspace, preflight, Telegram/Obsidian handoff 흐름을 하나의 작업실처럼 묶는 데 초점을 둡니다.
