## Private beta invite message
안녕하세요! Hermes for Web private beta 를 열었습니다.

이 프로젝트는 단순한 AI 채팅창이 아니라,
- 사용자의 Hermes memory 를 읽고
- Setup Packs 로 환경을 시작하고
- Artifact 로 결과물을 남기고
- Telegram / Obsidian / CLI 흐름까지 이어주는
개인화 AI 작업실을 목표로 합니다.

이번 beta 에서는 특히 아래를 보고 싶습니다.
- 첫 실행이 이해하기 쉬운지
- Setup Packs 가 실제로 도움이 되는지
- Artifact / Workspace 흐름이 자연스러운지
- 전체 UI 가 너무 복잡하지 않은지

추천 첫 사용 순서:
1. 봇 이름 설정
2. Setup Pack 하나 실행
3. Artifact 하나 생성
4. Telegram handoff 또는 Share 흐름 사용

피드백은 “무엇이 좋았는지”보다
“어디서 멈칫했는지”가 더 소중합니다.
